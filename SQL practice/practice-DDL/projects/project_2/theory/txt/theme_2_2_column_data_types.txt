При создании таблицы с помощью команды CREATE TABLE в SQL,
определяются столбцы таблицы и их типы данных.

Вот некоторые общие типы данных, которые могут быть использованы
при определении столбцов:

1. INT (целое число): Представляет целочисленные значения,
например, 1, 10, -5.

2. VARCHAR(n) (переменная длина строки): Представляет строки
переменной длины, где "n" указывает максимальную длину строки.
Например, VARCHAR(50) означает, что максимальная длина строки
составляет 50 символов.

3. CHAR(n) (фиксированная длина строки): Представляет строки
фиксированной длины, где "n" указывает точную длину строки.
Например, CHAR(10) означает, что строка всегда будет состоять из
10 символов, даже если фактическое значение короче.

4. DECIMAL(p, s) (десятичное число): Представляет числа с
фиксированной точностью и масштабом. "p" указывает общее
количество цифр (представляет всю числовую часть и дробную часть),
а "s" указывает количество цифр после десятичной точки.

5. DATE (дата): Представляет дату, например, '2023-05-31'.

6. TIME (время): Представляет время, например, '14:30:00'.

7. BOOLEAN (логическое значение): Представляет булево значение
true или false.

Это лишь некоторые примеры типов данных, которые могут быть
использованы при определении столбцов. В SQL существуют и
другие типы данных, которые могут быть использованы в зависимости
от конкретных потребностей и типа базы данных.

Вот несколько примеров создания столбцов с разными типами данных в SQL:

1. Целочисленный столбец:
```sql
   CREATE TABLE employees (
     id INT,
     name VARCHAR(50),
     age INT,
     salary DECIMAL(10, 2)
   );
```

2. Строковый столбец переменной длины:
```sql
   CREATE TABLE products (
     id INT,
     name VARCHAR(100),
     description VARCHAR(255)
   );
```

3. Строковый столбец фиксированной длины:
```sql
   CREATE TABLE customers (
     id INT,
     name CHAR(50),
     email CHAR(100)
   );
```

4. Столбец с десятичным числом:
```sql
   CREATE TABLE orders (
     id INT,
     order_number VARCHAR(10),
     total_amount DECIMAL(10, 2)
   );
```

5. Столбец с датой:
```sql
   CREATE TABLE tasks (
     id INT,
     task_name VARCHAR(50),
     due_date DATE
   );
```

6. Столбец с временем:
```sql
   CREATE TABLE appointments (
     id INT,
     client_name VARCHAR(50),
     appointment_time TIME
   );
```

7. Столбец с логическим значением:
```sql
   CREATE TABLE flags (
     id INT,
     flag_name VARCHAR(50),
     is_active BOOLEAN
   );
```

Это всего лишь некоторые примеры создания столбцов с разными
типами данных. Фактические типы данных и их размеры могут
варьироваться в зависимости от конкретных требований вашего
проекта и используемой СУБД (системы управления базами данных).
