Цель данного проекта заключается в освоении использования
операторов CREATE DATABASE и DROP DATABASE в языке SQL для
создания и удаления базы данных.

Ключевые моменты для достижения этой цели включают:

1. Понимание оператора CREATE DATABASE: Изучите синтаксис
оператора CREATE DATABASE и его функциональность для создания
новой базы данных. Понимание различных аспектов оператора
CREATE DATABASE, таких, как имя базы данных и дополнительные параметры,
поможет вам правильно создавать базы данных.

2. Создание базы данных: Практикуйтесь в написании запросов
CREATE DATABASE для создания новых баз данных. Узнайте, как указывать
имя базы данных и дополнительные параметры, если таковые есть,
чтобы создать базу данных с требуемыми характеристиками.

3. Понимание оператора DROP DATABASE: Изучите синтаксис оператора
DROP DATABASE и его функциональность для удаления базы данных.
Понимание правил и ограничений оператора DROP DATABASE поможет
вам безопасно удалять базы данных при необходимости.

4. Удаление базы данных: Практикуйтесь в написании запросов
DROP DATABASE для удаления баз данных. Убедитесь, что вы понимаете
последствия удаления базы данных и обязательно создавайте резервные
копии данных перед выполнением оператора DROP DATABASE.
