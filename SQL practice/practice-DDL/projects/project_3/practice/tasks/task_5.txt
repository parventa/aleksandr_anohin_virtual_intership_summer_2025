Напишите SQL команду для создания в базе данных
уникального индекса для названия и директора фильма из таблицы movies.
(воспользуйтесь SQL командой CREATE INDEX).

После создания уникального индекса, убедитесь в правильной его работе.
Попробуйте сохранить в базу данных дубликаты значений для уникального
индекса и убедитесь, что база данных выдаёт ошибку.
