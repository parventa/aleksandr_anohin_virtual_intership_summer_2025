Цель данного проекта заключается в изучении операторов
CREATE INDEX, DROP INDEX и UNIQUE INDEX в языке SQL для создания,
удаления и управления индексами в базе данных.

Ключевые моменты для достижения этой цели включают:

1. Понимание индексов в базах данных: Изучите, что такое индексы
и как они улучшают производительность запросов. Понимание принципов
работы индексов поможет вам оптимизировать работу с базой данных.

2. Создание индексов с использованием оператора CREATE INDEX:
Изучите синтаксис оператора CREATE INDEX и его функциональность
для создания индексов в базе данных. Практикуйтесь в создании
индексов для столбцов, которые часто используются в запросах,
чтобы повысить производительность запросов.

3. Удаление индексов с использованием оператора DROP INDEX:
Изучите синтаксис оператора DROP INDEX и его функциональность
для удаления индексов из базы данных. Практикуйтесь в удалении
ненужных индексов, чтобы улучшить производительность и управляемость
базы данных.

4. Уникальные индексы с использованием UNIQUE INDEX:
Изучите концепцию уникальных индексов и их использование для
обеспечения уникальности значений в столбцах. Практикуйтесь в
создании уникальных индексов, чтобы гарантировать уникальность
данных в соответствующих столбцах.

5. Расширение знаний (Анализ и оптимизация индексов):
Изучите методы анализа и оптимизации производительности
индексов в базе данных. Понимание различных факторов,
таких как размер индексов, фрагментация и статистика,
поможет вам оптимизировать работу с индексами и улучшить
производительность запросов.
