**Проект: "Система управления международным авиаперевозчиком"**

**Описание:**

Цель этого проекта - разработать базу данных для авиакомпании,
предоставляющей международные перевозки. Эта база данных должна
включать информацию о пассажирах, сотрудниках авиакомпании,
воздушном флоте, маршрутах, рейсах, билетах и информацию о багаже.

**Основные этапы проекта:**

1. **Проектирование базы данных:**
Создайте ER-диаграмму, которая отображает отношения между
сущностями "Пассажиры", "Сотрудники", "Воздушный флот",
"Маршруты", "Рейсы", "Билеты" и "Багаж".

2. **Создание базы данных:**
Используйте SQL DDL для создания базы данных.
Она должна содержать следующие таблицы:
   - Таблица "Пассажиры": ID пассажира, Имя, Фамилия,
     Дата рождения, Контактная информация.
   - Таблица "Сотрудники": ID сотрудника, Имя, Фамилия,
     Должность, Контактная информация.
   - Таблица "Воздушный флот": ID самолета, Модель,
     Год выпуска, Количество мест.
   - Таблица "Маршруты": ID маршрута, Пункт отправления, Пункт назначения.
   - Таблица "Рейсы": ID рейса, ID маршрута, ID самолета,
     ID капитана, Дата и время вылета, Дата и время прилета.
   - Таблица "Билеты": ID билета, ID рейса, ID пассажира,
     Класс обслуживания (эконом, бизнес), Стоимость.
   - Таблица "Багаж": ID багажа, ID пассажира, ID рейса, Вес багажа.

3. **Создание внешних ключей и индексов:**
Убедитесь, что все отношения между таблицами правильно
заданы с использованием внешних ключей. Также создайте
индексы для улучшения производительности запросов.

4. **Создание ограничений:**
Разработайте и примените ограничения целостности данных,
такие как CHECK (например, вес багажа не может быть отрицательным),
UNIQUE и NOT NULL.

**Ожидаемые результаты:**

В результате у вас должна быть создана база данных,
которую можно использовать для управления авиакомпанией.
База данных должна обеспечивать сохранность и целостность данных,
а также позволять быстро и эффективно искать нужную информацию,
например, информацию о рейсах, билетах или багаже.


Напишите миграционный скрипт с SQL командами,
который создаёт базу данных с нуля.

Данный скрипт должен включать в себя как минимум следующие команды:
- CREATE DATABASE;
- CREATE TABLE (+ PRIMARY KEY, FOREIGN KEY, NOT NULL);
- ALTER TABLE ADD COLUMN;
- CREATE INDEX (UNIQUE INDEX);
- CREATE VIEW;
- CREATE ROLE;
- CREATE USER;
- GRANT;

Продемонстрируйте возможности сохранения данных в созданную вами
структуру базы данных при помощи SQL скрипта с командами вставки данных.

Создайте в базе данных набор VIEW, которые помогут бизнесу с нахождением
ответом на типичные запросы и анализом данных хранящихся в базе данных.
