Команда CREATE ROLE в базе данных используется для создания роли,
которая является группой пользователей с общими правами доступа.

Давайте представим, что у нас есть база данных для онлайн-магазина,
и мы хотим создать роль "модератор". Роль "модератор" будет иметь
право на редактирование и удаление отзывов о товарах.

Пример команды CREATE ROLE для создания роли "модератор" в базе данных:

```sql
CREATE ROLE moderator;
```

После выполнения этой команды, роль "модератор" будет создана.
Теперь мы можем назначать эту роль определенным пользователям.

Пример назначения роли "модератор" для пользователя "john" в базе данных:

```sql
GRANT moderator TO john;
```

Теперь пользователь "john" получил роль "модератор" и все связанные
с ней права доступа. Это означает, что он может редактировать и
удалять отзывы о товарах в базе данных.

Команда CREATE ROLE позволяет создавать роли и определять
их набор прав доступа. Это удобный способ группировать
пользователей и управлять их правами в базе данных.
